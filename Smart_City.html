Smart City Large Language Model Agent System

專案介紹

本計畫由國立臺北大學團隊攜手業界共同推動，致力於打造一個先進的智慧城市大型語言模型代理人系統。該系統整合最先進的語言模型技術（採用 Llama 3.2: 3B 模型），並結合交通部開源 API 提供的即時路況資訊，旨在為使用者提供快速、準確的交通查詢服務。

背景與動機

隨著智慧城市概念普及及都市化進程加速，市民對交通狀況即時掌握與智慧服務的需求日益增加。本計畫結合先進語言模型與即時交通數據，提供精準的交通查詢與智能回應，進一步推動智慧城市建設。

系統功能

  • 即時交通查詢： 系統能即時解析使用者查詢，並透過 API 獲取最新交通資訊。
  • 先進語言模型： 採用 Llama 3.2 模型，提供自然語言理解與生成，回應更智慧與人性化。
  • 穩健系統架構： 後端使用 Flask 框架，結合 Docker 與 Google Cloud Run 部署，確保高效能與高可用性。
  • 未來擴充： 未來計畫引入 GPS 定位及更多個性化服務，以提升使用體驗。

市場分析

目前智慧城市相關解決方案多停留在基礎資訊查詢層面，缺乏智慧互動與全面數據整合。本計畫利用先進語言模型與即時數據，滿足市民對智慧服務的需求，並在市場中展現競爭優勢。

預期效益

本計畫預計將提升交通資訊服務品質、促進智慧城市建設，並推動相關產業的數位轉型。
