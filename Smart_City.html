III x NTPU AI Agent Project

Smart City Large Language Model Agent System

專案介紹

本計畫由國立臺北大學的團隊攜手業界共同推動，致力於打造一個先進的智慧城市語言模型代理人系統。該系統結合最新的大型語言模型技術（採用 Llama 3.2: 3B 模型），並整合交通部開源 API 提供的即時路況資訊，旨在為使用者帶來快速、準確的交通查詢服務。

背景與動機

隨著智慧城市概念的普及與都市化進程加速，市民對交通狀況的即時掌握與智慧服務需求日益增加。本計畫旨在整合先進語言模型技術與即時交通數據，提供精準的交通查詢與智能回應，進一步推動智慧城市建設。

系統功能

  • 即時交通查詢與資料整合： 系統能夠即時解析使用者的查詢，並透過 API 取得最新的交通資料，回應查詢需求。
  • 先進的大型語言模型： 採用 Llama 3.2 模型，提供自然語言理解與生成，使回應更具智慧與人性化。
  • 穩健的系統架構： 後端使用 Flask 框架搭建，結合 Docker 容器化技術與 Google Cloud Run 部署，確保高效能與高可用性。
  • 未來擴充性： 未來將引入 GPS 定位與更多個性化服務，進一步提升使用體驗與應用價值。

市場分析

目前智慧城市相關解決方案多停留在基礎資訊查詢層面，缺乏智慧互動與全面數據整合。本計畫利用先進的語言模型與即時交通數據，能夠滿足市民對智慧城市服務的需求，並在市場中展現明顯競爭優勢。

預期效益

計畫成果預計將於2024年11月7日在國立臺北大學展示，由戴敏育教授領導，團隊成員包括盧信廷、籃勗祐、陳柏臻。此系統不僅能提升交通資訊服務品質，還將作為智慧城市建設的示範，推動相關產業的數位轉型與創新發展。
