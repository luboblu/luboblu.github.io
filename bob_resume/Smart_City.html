III x NTPU AI Agent Project

Smart City Large Language Model Agent System

智慧城市大型語言模型代理人系統

計畫介紹

本計畫由國立臺北大學的團隊攜手業界共同推動，致力於打造一個先進的智慧城市語言模型代理人系統。該系統結合最新的大型語言模型技術（採用 Llama 3.2: 3B 模型），並整合交通部開源 API 提供的即時路況資訊，旨在為使用者帶來快速、準確的交通查詢服務。

系統特色與架構

  • 即時交通查詢與資料整合： 系統能夠即時解析使用者的查詢，並透過 API 取得最新的交通資料，進而回應查詢需求。
  • 先進的大型語言模型： 採用前沿的 Llama 3.2 模型，提供自然語言理解與生成，讓查詢回應更具智慧與人性化。
  • 穩健的系統架構： 後端以 Flask 框架搭建，並利用 Docker 容器化技術與 Google Cloud Run 部署，確保系統具有高效能與高度可用性。
  • 未來擴充性： 除了現有功能，系統未來將引入 GPS 定位與更多個人化服務，進一步提升使用體驗與實際應用價值。

團隊與成果展示

計畫成果於 2024 年 11 月 7 日在國立臺北大學展示，由戴敏育教授領導，團隊成員包括多位研究助理（盧信廷、籃勗祐、陳柏臻）。不論在技術實現或系統設計上，皆展現了學術與產業緊密結合的優勢，為智慧城市未來的交通管理與服務提供全新的解決方案。
